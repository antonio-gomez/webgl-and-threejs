What is WebGL?

(web graphics library)

Is a JavaScript API for rendering interactive 2D and 3D computer graphics within any compatible web browser without the use of plug-ins.

We don't need third-party plug-ins to extend the web browser capabilities.

Major browser vendors like Apple (Safari), Google (Chrome), Mozilla (Firefox), and Opera (Opera) are members of the WebGL Working Group.

Gradually becoming an standard.

(for desktop and mobile devices)
>

While a simple cube in raw WebGL would turn out hundreds of lines of JavaScript and shader code...

An easy to use library created by Ricardo Cabello
(Mr. doob) and maintained by the GitHub community, can achieve robust 3D simulations with a fraction of code.

Giving us the flexibility to create and manipulate:

geometry, lights, scenes, materials and particles.
